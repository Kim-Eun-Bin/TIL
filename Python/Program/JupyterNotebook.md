## mac에서 Jupyter notebook 설치 및 실행


jupyter notebook을 설치하기 전에 업데이트를 먼저 실행합니다.
```
>> pip3 install --upgrade pip
```

pip3로 jupyter를 설치합니다.
```
>> pip3 install jupyter
```

jupyter notebook 실행
```
>> jupyter notebook
```
